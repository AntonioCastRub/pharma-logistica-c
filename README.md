# pharma-logistica-c
La PYME “La Farmacéutica” integra verticalmente su cadena de suministro. En el área de TICs se solicita un sistema modular que gestione un pedido logístico desde la captura hasta el reporte, aplicando funciones, paso de parámetros por valor y por referencia, ámbito de variables y sentencias de control (if–else, switch, for, while, do–while).
